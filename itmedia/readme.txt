■task01
問題1)
引数として与えられた半角アルファベットの文字列(文字列長1〜15文字とする)について、大文字と小文字を反転させ、逆順にした文字列を表示するプログラムを作成してください。
例) ITmedia → AIDEMti

・実行方法
引数に変換したい文字列を指定して実行する
例) task01 ITmedia


■task02
引数として与えられた数値 N 番目の素数(1 より大きい自然数で正の約数が 1 と自分自身のみである数 例 2, 3, 5, 7,
11…)を表示するプログラムを作成してください。
例) 3 → 5

・実行方法
引数に数値 N 番目の素数を指定して実行する
例) task02 ITmedia
